# Resume

[View PDF](https://asmacdo.github.io/resume/resume.pdf)

Built with [tectonic](https://tectonic-typesetting.github.io/), deployed to GitHub Pages on push.

## Build locally

Install tectonic ([more options](https://tectonic-typesetting.github.io/en-US/install.html)):
```bash
conda install -c conda-forge tectonic
```

Then:
```bash
make
```
